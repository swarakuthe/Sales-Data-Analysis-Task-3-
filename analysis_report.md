# Sales Data Analysis Report

# 1. Project Title

Sales Data Analysis Using Python

# 2. Objective

The objective of this project is to analyze a sales dataset to calculate revenue, identify best-selling products, and generate useful business insights.

# 3. Tools Used

Python

Pandas

# 4. Data Cleaning Process

Checked for missing values

Filled missing Quantity values with 0

Filled missing Price values with average price

Removed duplicate rows

# 5. Feature Engineering

Created a new column Revenue using:
Revenue = Quantity × Price

# 6. Key Metrics Calculated

Total Revenue

Total Quantity Sold

Best Product by Quantity

Best Product by Revenue

Top 3 Products by Revenue

Highest Single Sale

Lowest Single Sale

Total Unique Products

Average Revenue per Product
